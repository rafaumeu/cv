## Overview

An interactive developer resume/CV application with **internationalization (PT/EN)**, built with **React**, **Vite**, and **TailwindCSS**. Features a modern design with responsive layout and smooth animations.

## Features

- Bilingual support (Portuguese/English)
- Responsive design for all devices
- Print-optimized layout
- Smooth scroll navigation
- Dark/light section themes

## Tech Stack

| Technology | Purpose |
|---|---|
| **React** | UI library |
| **TypeScript** | Type safety |
| **Vite** | Build tool |
| **TailwindCSS** | Styling |
| **React Intl** | Internationalization |
| **Lucide Icons** | Icon library |

## Getting Started

```bash
git clone https://github.com/rafaumeu/cv.git
cd cv
npm install
npm run dev
```
